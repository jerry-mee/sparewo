# Lottie Animations for SpareWo

## Required Animations

1. **car_repair.json** - Home screen welcome animation
   - Download from: https://lottiefiles.com/53736-car-repair
   
2. **loading_parts.json** - Loading state for product lists
   - Download from: https://lottiefiles.com/36605-car-services

3. **cart_bounce.json** - Cart icon animation when adding items
   - Download from: https://lottiefiles.com/88453-add-to-cart

4. **search.json** - Search bar focus animation
   - Download from: https://lottiefiles.com/72883-search-files

5. **welcome.json** - Onboarding welcome animation
   - Download from: https://lottiefiles.com/99297-welcome

## How to Add Animations

1. Download the JSON files from the links above
2. Place them in the `assets/animations/` directory
3. The app will automatically use them

## Fallback Images

If animations fail to load, the app will fall back to static images or icons.
